Detailed Description

template<typename T, typename ELEM_COMPARE = MemCmp<T>>
class shad::Set< T, ELEM_COMPARE >

The Set data structure.

SHAD's set is a distributed, unordered, set.

Template Parameters
Ttype of the entries stored in the set.
ELEM_COMPAREelement comparison function; default is MemCmp<T>.
Warning
obects of type T need to be trivially copiable.
